# 🌎 Localiza Já

Um projeto simples e útil que, a partir de um CEP, mostra informações completas sobre o local.

### ✨ Funcionalidades

- Busca de endereço completo pelo CEP
- Mapa interativo da localização
- Clima atual da cidade
- Números de emergência da região
- Design responsivo (funciona bem no celular)

### 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- APIs públicas:
  - ViaCEP (endereço)
  - Open-Meteo (clima)
  - Google Maps Embed

### 🎯 Objetivo do projeto

Praticar consumo de APIs, manipulação do DOM e criação de uma interface limpa e funcional.

### 📱 Como usar

1. Digite qualquer CEP válido
2. Clique em "Buscar"
3. Veja todas as informações na tela
